# 🖼️ Image Annotation Tool for Computer Vision

## 📌 Overview

Image Annotation Tool is a user-friendly application designed for labeling images by drawing bounding boxes around objects. It helps create high-quality datasets for Computer Vision and Object Detection models, making the data preparation process faster and more efficient.

---

## ✨ Features

* 📷 Upload images for annotation
* 🟦 Draw and edit bounding boxes
* 🏷️ Label objects for dataset creation
* 💾 Save annotation data
* ⚡ Fast and intuitive interface
* 🤖 Ideal for AI and Machine Learning projects

---
